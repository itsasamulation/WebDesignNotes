HTML images

HTML images are included in web pages using the image element. The syntax for adding images is included in the code of this page.

Syntax = <img src="image-path" alt="alternative-text" width ="300" height = "200">

  • 'src' can be a relative or aboslute path to the image file or a URL
  • 'alt'attribute provides alternative text for the image.
  • width & height represents pixels

File Formats

The <img> element in HTML can use various image file formats.

JPEG (Joint Photographic Experts Group)

  • Suitable for photographs and images with gradients.
  • Supports lossy compression, which can result in smaller file sizes

PNG (Portable Network Graphics)

  • Suitable for images with transparency or where lossless compression is preferred

GIF (Graphics Interchange Format)

  • Supports animations (multiple frames).
  • Limited to 256 colors, making it suitable for simple graphics and logos.

SVG (Scalable Vector Graphics)

  • A vector image format that is resolution-independent and can be scaled without loss of quality.
  • Suitable for logos, icons, and simple graphics.

BMP (Bitmap)

  • Generally not recommended for the web due to large file sizes.
  • Uncompressed format that retains high-quality detail.

WebP

  • Developed by Google, WebP is a modern image format that supports both lossy and lossless compression.
  • Provides high-quality images with smaller file sizes compared to JPEG and PNG

AVIF (AV1 Image File Format)

  • open, royalty-free video coding format
  • Efficient compression of high-quality images
