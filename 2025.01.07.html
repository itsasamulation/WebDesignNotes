Writing comments in the code

You can use comments to add notes either for yourself or for other designers/developers you might be working with. To add a comment press CMD/CTRL + /

Comments do not apply any formatting, spacing, or have any impact on the display.

You can use comments to easily hide elements from the page when you are editing. When you are dealing with a long document, this is an easy way to hide a big element.
