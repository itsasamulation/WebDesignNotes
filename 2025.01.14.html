HTML Links

A hyperlink is a clickable element in web pages or electronic documents, that directs users to another location when clicked.

It's a way to connect different parts of a document or link to external resources like other web pages, files, images, or even specific sections within the same page.

The title attribute specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element.

The target attribute specifies where to open the linked document.

  • _self opens the document in the same window/tab as it was clicked
  • _blank opens the document in a new window or tab
  • _parent opens the document in the parent frame
  • _top opens the document in the full body of the window

Absolute and Relative Hyperlinks

Absolute Hyperlinks contain the complete URL or web address of the linked resource. This is not the best link to use within a website.

Relative Hyperlinks specify the path to the linked resource relative ot the current location of the HTML file. They are often used when linking to pages within the same website.

Linking to folders and sub-folders

  • / Root directory
  • . This location
  • .. Up a directory
  • ./ Current directory
  • ../ Parent of current directory
  • ../../ Two directories backwards
